# Identificador-TAG
Automação em JavaScript para auxiliar os primeiros contatos com HTML, trazendo etiquetas e bordas identificadoras para as tags presentes no código

# Como adicionar em seu projeto?
Baixe o arquivo na pasta do seu projeto, e no html, no final da tag body, adicione a tag:

<script src="automacao.js"></script>

ATENÇÃO! - mude a parte do "automacao.js" de acordo com o caminho do seu projeto.

Seja feliz!
